# Gestión de lockers mediante datos biométricos

## Especificación de requisitos de software

### Introducción
Se definen los requerimientos funcionales y no funcionales para el sistema de lockers inteligentes con autenticación biométrica.

El sistema optimiza la asignación y liberación de lockers reemplazando llaves por huella dactilar.

---

## Propósito
Establecer una base clara para todos los participantes del proyecto y definir los requisitos del sistema.

Dirigido a:
- Equipo de desarrollo
- QA
- Gerentes
- Clientes
- Soporte

---

## Alcance
Sistema con:
- Interfaz de usuario
- Módulo administrador
- Lector de huella

Funciones:
- Automatizar préstamo de lockers
- Mejorar seguridad
- Reducir tiempos
- Llevar registro
